# 🧠 AI Image Detector

## 📌 Project Overview

This project is a Deep Learning-based system that detects whether an image is **Real** or **AI-generated (Fake)** using Convolutional Neural Networks and Transfer Learning.

---

## 🚀 Features

* Detects Real vs AI-generated images
* Uses MobileNetV2 (Transfer Learning)
* Streamlit Web Application
* Real-time prediction with confidence score

---

## 🛠️ Tech Stack

* Python
* TensorFlow / Keras
* Streamlit
* NumPy
* PIL

---

## 📊 Model Details

* Architecture: MobileNetV2 + Dense Layers
* Input Size: 128x128
* Output: Binary Classification (Real / Fake)

---

## 📁 Dataset

Dataset used:
Real vs Fake Image Dataset (Kaggle)

---

## ▶️ How to Run

### Step 1: Clone Repo

git clone https://github.com/your-username/ai-image-detector.git

### Step 2: Install Requirements

pip install -r requirements.txt

### Step 3: Run App

streamlit run app.py
